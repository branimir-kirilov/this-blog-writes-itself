## What is LangChain?

LangChain is a library designed to simplify the development of language model applications. It provides a framework for chaining together different components required to build complex language models, making it easier to manage and scale AI solutions.

### Key Features of LangChain

- **Modular Design:** Allows developers to build applications using interchangeable components.
- **Integration Ready:** Seamlessly integrates with various AI models, including OpenAI.
- **Customization:** Offers flexibility in customizing each component to cater to specific needs.

## Why OpenAI?

## Integrating LangChain and OpenAI

### Installing Dependencies

First, install the necessary packages:

```bash
npm install langchain openai
```

### Configuring OpenAI

Create a new file named `.env` in the root directory of your project. Add your OpenAI API key:

```
REACT_APP_OPENAI_API_KEY=your_openai_api_key
```

### Setting Up LangChain

Within your `src` directory, create a new file `LangChainConfig.js`:

```javascript
import { LangChain } from 'langchain';

const lc = new LangChain({
  apiKey: process.env.REACT_APP_OPENAI_API_KEY,
});

export default lc;
```

## Building the Chatbot Component

Create a new component `Chatbot.js`:

```javascript
import React, { useState } from 'react';
import lc from './LangChainConfig';

const Chatbot = () => {
  const [input, setInput] = useState('');
  const [messages, setMessages] = useState([]);

  const sendMessage = async () => {
    const response = await lc.generateResponse(input);
    setMessages([...messages, { role: 'user', content: input }, { role: 'bot', content: response }]);
    setInput('');
  };

  return (
    <div>
      <div>
        {messages.map((msg, index) => (
          <p key={index}><strong>{msg.role}:</strong> {msg.content}</p>
        ))}
      </div>
      <input value={input} onChange={(e) => setInput(e.target.value)} />
      <button onClick={sendMessage}>Send</button>
    </div>
  );
};

export default Chatbot;
```
